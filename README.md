# Smart Wallet — Expense Tracker

A cross-platform mobile app built with Flutter and Dart that helps users track personal finances, manage expenses, and visualize spending habits in real time.

Developed as a senior capstone project at the University of Texas Rio Grande Valley.

---

## Features

- Secure sign-up and login via Firebase Auth
- Add, categorize, and manage income and expense transactions
- Real-time cloud sync with Cloud Firestore
- Interactive spending charts and breakdowns
- Profile photo upload via camera or gallery
- Onboarding flow for new users
- Native splash screen on launch

---

## Tech Stack

- **Framework:** Flutter / Dart
- **Auth & Database:** Firebase Auth, Cloud Firestore, Firebase Storage
- **State Management:** GetX
- **Charts:** fl_chart
- **Camera:** camera, image_picker
